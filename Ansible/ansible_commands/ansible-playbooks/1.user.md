# Create a user playbook

```sh
vi user.yaml
```
```yaml
---
- name: create a user
  hosts: all
  become: true
  tasks:
   - name: create user
     user:
        name: alexa
...
```
```sh
ansible-playbook user.yaml
```
Verify if users are created in the nodes 
```sh
cat /etc/passwd
```
OR
```sh
ansible all -a "cat /etc/passwd"
```
