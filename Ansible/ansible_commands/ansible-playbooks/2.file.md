# Create a file playbook

```sh
vi file.yaml
```
```yaml
---
- name: create a file
  hosts: all
  become: true
  tasks:
   - name: create file module
     file:
        path: /tmp/file1
        state: touch
...
```
```sh
ansible-playbook file.yaml
```
Verify file creation in Master
```sh
ansible all -a “ls /tmp”
```
