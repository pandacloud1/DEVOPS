# Copy Module
Playbook to copy file from Master --> Node and with change of owner, group & permission
```sh
vi copy.yaml
```
```yaml
---
- name: copy module
  hosts: all
  become: true
  tasks:
  - name: copy files
    copy:
      src: /tmp/file1.txt
      dest: /tmp/file1.txt
      owner: alexa
      group: amazon
      mode: 0744
...
```
