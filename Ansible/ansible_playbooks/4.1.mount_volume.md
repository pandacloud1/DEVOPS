# Mount volumes in Nodes
```sh
vi mount.yaml
```
```yaml
---
- name: mount volume
  hosts: all
  become: true
  tasks:
  - name: create mount point (dir)
    file: 
      path: /mnt/data1
      state: directory
      mode: '0755'

  - name: create filesystem
    filesystem: 
      fstab: ext4
      dev: /dev/sdb1		# dev: device

  - name: mount filesystem
    mount: 
      src: /dev/sdb1
      path: /mnt/data1
      fstype: ext4
      state: mounted
      state: 
...
```
```sh
ansible-playbook mount.yaml
```
